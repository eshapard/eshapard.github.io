Saving A Game Transcript In Gargoyle

Posted on April 22, 2021

Yeah, so this is sort of embarrassing. I play interactive fiction using Gargoyle every now and then, and I’ve been trying to figure out a way around Gargoyle’s obvious lack of any transcript-export feature.

Turns out, you just have to type script into the parser. From then on, all the text and commands get written to a file (that you select). It’s built-in, like save or restore.

Who knew?
